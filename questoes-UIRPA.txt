✅ Pergunta 5
Boas práticas ao automatizar um projeto: Selecione todas as respostas aplicáveis.

✔️ Respostas corretas:

Usar nomes significativos para arquivos de workflow, atividades, argumentos e variáveis

Escolher o layout apropriado (fluxogramas, sequências, máquinas de estado)

Usar sistema de controle de versão

Deletar arquivos de saída no início do processo

Usar convenção Upper Camel Case para nomear workflows

✅ Pergunta 6
Como podemos pausar propositalmente o processo de depuração em uma atividade que pode gerar problemas de execução? Escolha uma das respostas.

✔️ Resposta correta:

Clicando com o botão direito na atividade e selecionando Toggle Breakpoint

✅ Pergunta 7
Como publicar no 'Orchestrator Personal Workspace Feed'? Escolha uma das respostas.

✔️ Resposta correta:

Primeiro habilitando a opção no Orchestrator e depois selecionando 'Orchestrator Personal Workspace Feed' ao publicar no Studio

✅ Pergunta 8
Qual atividade do Studio usamos para repetir ações em cada arquivo de uma pasta? Escolha uma das respostas.

✔️ Resposta correta:

For each file in folder

✅ Pergunta 9
Como abrir a localização do projeto na máquina usando o Studio? Escolha uma das respostas.

✔️ Resposta correta:

Clicando no painel Project e selecionando File Explorer

✅ Pergunta 10
Quando devemos usar a atividade 'If' em um workflow no Studio? Escolha uma das respostas.

✔️ Resposta correta:

Quando queremos que o processo siga um de dois caminhos diferentes, dependendo de uma condição

✅ Pergunta 11
Como adicionar uma atividade ao workflow no Studio? Escolha uma das respostas.

✔️ Resposta correta:

Clicando no painel Activities, selecionando ou pesquisando a atividade e arrastando para o workflow

✅ Pergunta 12
Como executar o projeto em modo de depuração no Studio? Escolha uma das respostas.

✔️ Resposta correta:

Clicando na aba Debug, depois em Debug File e selecionando Run File
